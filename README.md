## Deploy To VPS
`git clone https://github.com/AshutoshGoswami24/text-leech-bot`

`cd text-leech-bot`

`pip install -r requirements.txt`

`sudo apt update`

`sudo apt install ffmpeg`

`python3 modules/main.py`
